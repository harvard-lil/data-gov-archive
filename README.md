# Data.gov Archive Search

A static discovery layer for the Library Innovation Lab's [Data.gov Archive](https://source.coop/harvard-lil/gov-data). Uses [DuckDB-Wasm](https://duckdb.org/2021/10/29/duckdb-wasm.html) to execute client-side queries of archive catalog metadata.

For more information, see the [announcement on our blog](https://lil.law.harvard.edu/blog/2025/10/10/welcome-to-lil-s-data-gov-archive-search/).

## Quickstart

```sh
# Install dependencies
npm install

# Run in development mode
npm run dev

# Build for production
npm run build
```
