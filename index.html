This interface supports searching the Data.gov Archive, a collection of 311,604 datasets (approximately 17.9 TB) harvested between November 2024 and January 2025.

Data.gov Archive Search is a project of the Library Innovation Lab (LIL) at Harvard Law School. For more information on our work, see LIL's Public Data Project.
